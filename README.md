# FrameFunRBF
A julia package extending the Basisfunctions package with a dictionary consisting of RBFs.

## Installation
```julia
using Pkg
pkg"add registry https://github.com/FrameFunVC/FrameFunRegistry"
pkg"add FrameFunRBF"
```
